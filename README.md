**Author:** Joannalyn Delacruz, PhD

---

## Abstract

The Biophysiological Feedback Framework (BFF) defines a modality-agnostic architecture for observing interaction patterns and updating structured belief states representing readiness, regulation, persistence, and consistency.

The framework separates surface interaction from assessment logic through a three-layer architecture. BFF does not perform diagnosis, therapy, or normative comparison. It produces within-individual descriptive indicators derived from structured event streams.

This document describes conceptual constructs and architectural boundaries. Implementation details, scoring weights, and calibration logic are maintained separately.

---

## 1. Problem Statement

Digital interaction systems frequently conflate:

- Surface rendering  
- Interaction capture  
- Metric computation  
- Behavioral interpretation  

This coupling reduces extensibility and compromises interpretability.

BFF addresses this by defining:

- A modality-agnostic interaction representation  
- A state-based inference engine  
- Conservative policy gating to prevent over-interpretation  

The objective is not to classify users, but to describe patterns of interaction over time within individuals.
